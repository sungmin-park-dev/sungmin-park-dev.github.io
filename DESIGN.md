# emergence - 디자인 철학

## 브랜드 아이덴티티

**emergence** (창발성)는 다체계 물리학의 핵심 개념입니다. 단순한 요소들의 상호작용에서 예측할 수 없었던 복잡하고 아름다운 현상이 나타나는 것 — 이것이 emergence입니다.

이 포트폴리오는 물리학과 AI의 경계에서 연구하는 연구자의 여정을 담습니다. 양자 다체계, 텐서 네트워크, 그리고 머신러닝이 만나는 지점에서 새로운 통찰이 창발하는 순간을 기록합니다.

## 듀얼 테마 시스템

사이트는 다크/라이트 두 가지 테마를 지원합니다. 우측 상단 토글로 전환하며, 각 테마는 독립적인 컨셉을 가집니다.

### 다크 모드 — Stellar Blue (항성광)

- **배경**: 인디고 블랙 (`#010306 → #030812`) — 우주 심연, OLED 최적화
- **강조색**: 청백색 항성광 (`rgba(0, 200, 255, 0.92)`) — 어둠 속 빛나는 아이디어
- **의미**: 광활한 우주에서 빛을 발하는 항성의 빛처럼, 깊은 어둠 속에서 지식이 창발하는 순간

### 라이트 모드 — Deep Glacier (심층 빙하)

- **배경**: 빙하 청색 틴트 (`#EEF5FA → #F8FCFF`) — 얼음의 순수함, 명료함
- **강조색**: 심층 빙하 청색 (`#1560A8`) — 빙하 깊은 곳의 압축된 파란빛
- **의미**: 빙하가 수천 년의 압력으로 만들어낸 순수한 파란빛처럼, 집약된 사유의 명료함

### 색상 토큰

모든 색상은 CSS 커스텀 프로퍼티로 정의되어 런타임에 전환됩니다.
SCSS에 하드코딩 없이 `_data/theme_dark.yml` / `_data/theme_light.yml` 에서 관리합니다.

## 디자인 컨셉

### 유리질감 (Glassmorphism)
- **기법**: `backdrop-filter: blur(10–15px)` + 반투명 배경
- **의미**: 다층적 구조, 겹쳐진 정보의 레이어
- **효과**: 깊이감과 공간감 — 정보가 공간에 떠 있는 듯한 몰입감

### 극도의 경량화 (Ultra-Light Typography)
- **폰트 두께**: 100–300 (Thin ~ Light)
- **의미**: 최소한의 시각적 노이즈, 본질에 집중
- **효과**: 애플 디자인 언어의 우아함과 절제

### 점(.)의 철학
"emergence." — 오직 타이틀에만 존재하는 점은 **완결성**을 상징합니다. 연구는 진행형이지만, 각 순간은 완결된 통찰입니다.

## 디자인 원칙

### 1. 창발성의 시각화
단순한 요소(카드, 타이포, 컬러)가 조화롭게 배치되어 전체가 부분의 합보다 큰 경험을 만듭니다.

### 2. 상호작용의 보상
모든 호버 효과는 의미 있는 변화를 줍니다.
- **Transform**: 살아있는 느낌 (`translateY(-4px)`)
- **Glow**: 주목의 표시 (accent color shadow)
- **Border**: 상태의 명확한 전환 (opacity 0.3 → 0.7)

### 3. 토큰 기반 테마
SCSS 변수가 아닌 CSS 커스텀 프로퍼티로 모든 색상을 관리합니다.
SCSS 지식 없이도 `_data/*.yml` 파일만 수정하면 전체 테마를 바꿀 수 있습니다.

### 4. 최소한의 의존성
외부 CSS 프레임워크 없이 순수 SCSS (~2,800줄)로 전체 디자인 시스템을 구현합니다.
Jekyll 외 자바스크립트 라이브러리 의존 없음.
